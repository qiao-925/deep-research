# 研究方法论

## 概述

本知识库中的每份报告均由 Claude Code 的 **Deep Research 工作流** 生成，采用混合研究法（量化+质性）和对抗性验证流程。

## 5阶段动态迭代流程

### Phase 1: 宏观图景与数据锚定

大规模数据扫描，建立基线事实：
- 官方统计时间序列（国家统计局、UNFPA等）
- 大型微观面板数据库（CGSS、CFPS、CHARLS等）
- 区域异质性标注

### Phase 2: 机制解剖与深层归因

从"环境倒逼观念"的视角切入，进行因果推断：
- 经济基础维度（城市化、教育、就业结构）
- 制度替代维度（社会保障体系）
- 隐性延续维度（观念与实践的张力）

### Phase 3: 代际断裂与微观叙事

引入时间维度，捕捉观念的动态流动：
- 世代效应（Cohort Effect）分析
- 决策画像构建

### Phase 4: 批判性反驳与反常识挖掘

对抗性自我纠偏：
- 反例寻找
- 政策悖论分析
- 全球横向对比

### Phase 5: 综合研判与趋势预测

基于全部发现的系统性判断：
- PSR模型构建
- 阶段性结论
- 未来趋势预测

## 对抗性验证机制

每份报告的核心主张均经过 **3轮独立验证**：

1. **提取**：从抓取的来源中提取可证伪的主张
2. **验证**：3个独立agent分别尝试反驳每个主张
3. **判定**：≥2/3否决 → 淘汰；否则保留

验证统计示例：
- 提取主张：58个
- 进入验证：25个
- 通过验证：11个（44%）
- 淘汰：14个（56%）

## 自动归档

每次 Deep Research 工作流完成后，报告自动归档到本知识库，包含：
- 完整研究报告（Markdown格式）
- 研究过程元数据（token消耗、agent数量、验证统计）
- 来源清单与质量标注
